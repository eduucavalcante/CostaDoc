Sistema simples de cadastro e gerenciamento de imóveis e clientes, e geração de contratos de locação e contratos de compra e venda, para escritório de contabilidade real.

**Tecnologias utilizadas:**
- Node.js
- Express
- MySQL
- JWT
- bcrypt
- docxtemplater
- pizzip
- Axios
- Sequelize
- CORS
- Dotenv
- HTML, CSS, JavaScript
